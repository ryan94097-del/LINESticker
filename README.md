# LINE 貼圖處理器

自動處理圖片成符合 LINE 規範的格式，包含貼圖分割與主要圖片/標籤圖片轉換功能。

## 功能

### 📐 貼圖分割
- 🔍 自動識別並分割每個獨立貼圖
- 📏 支援網格分割（推薦）與自動偵測兩種模式
- 🎨 AI 智慧去背 (使用 rembg)
- 📐 自動調整為 LINE 貼圖規格 (**370 x 320 px**)
- 📦 一鍵打包下載 ZIP

### 🖼️ 主要圖片/標籤圖片轉換
- 🖼️ **主要圖片 (main)**: 240 x 240 px
- 💬 **聊天室標籤圖片 (tab)**: 96 x 74 px
- 🎨 可選擇是否執行 AI 去背
- 📦 可同時輸出兩種尺寸

## LINE 貼圖尺寸規格

| 類型 | 尺寸 | 說明 |
|------|------|------|
| 貼圖圖片 | 370 x 320 px (最大) | 個別貼圖的尺寸 |
| 主要圖片 (main) | 240 x 240 px | 貼圖商店的預覽圖 |
| 聊天室標籤 (tab) | 96 x 74 px | 聊天室中的標籤圖示 |

## 部署到 Streamlit Cloud

### 步驟 1: 上傳至 GitHub

1. 在 GitHub 建立新儲存庫 (例如: `line-sticker-splitter`)
2. 上傳以下檔案:
   - `app.py`
   - `requirements.txt`

### 步驟 2: 部署

1. 前往 [Streamlit Cloud](https://streamlit.io/cloud)
2. 使用 GitHub 帳號登入
3. 點擊 **New app**
4. 選擇你的儲存庫和 `app.py`
5. 點擊 **Deploy**

> ⚠️ 首次部署可能需要 5-10 分鐘安裝依賴套件

## 使用方式

### 貼圖分割
1. 選擇「📐 貼圖分割」頁籤
2. 上傳包含多個貼圖的合集圖片 (JPG/PNG)
3. 選擇分割模式（網格分割或自動偵測）
4. 調整參數後點擊「開始處理」
5. 預覽結果並下載 ZIP

### 主要圖片/標籤圖片轉換
1. 選擇「🖼️ 主要圖片/標籤圖片」頁籤
2. 上傳要轉換的圖片 (JPG/PNG)
3. 選擇要輸出的尺寸類型
4. 點擊「開始轉換」
5. 下載處理完成的圖片

## 技術規格

- **貼圖輸出尺寸**: 370 x 320 px (LINE 貼圖最大規格)
- **主要圖片輸出尺寸**: 240 x 240 px
- **聊天室標籤輸出尺寸**: 96 x 74 px
- **輸出格式**: PNG (透明背景)
- **邊距**: 約 5-10px 透明邊框
